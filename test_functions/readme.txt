Второстепенные функции по иследованию БД и проверки работоспособности алгоритмов.
Среда моделирования GNU Octave

get_all_crop_obj - Читаем БД, автоматом ищем соседние кадры, формируем БД вырезанных объектов
add_obj_on_fon - Автоматом ищем подходящие кадры в качестве фона, накладываем случайные объекты из БД объектов на фон с аугментацией. Формируем новую БД для обучения
SDC - Проверка обнаружения движущихся целей и разрезания изображения на оптимальное количество фрагментов
make_video_from_dataset - Синтезируем видео. На реальный фон накладываем цели и имитируем их движение
